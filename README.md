## Projects

### Machine Learning & AI

**[Quant Trading AI](https://github.com/TungAnhNguyen5/quant_trading_ai)** — End-to-end ML pipeline for intraday trading on VN30 futures (Vietnamese derivatives market). Covers feature engineering from order book and tick data (OBI, OFI, CVD, VWAP), training classifiers (XGBoost, TabNet, TFT, LSTM, 1D-CNN, PPO), trailing-stop backtesting, and realtime inference via shared memory.
`Python` `PyTorch` `XGBoost` `pytorch-forecasting` `PostgreSQL` `Optuna`

**[BLIP-2 Explorer](https://github.com/TungAnhNguyen5/blip2-explorer)** — Fine-tuned BLIP-2 for underwater marine image captioning using LoRA adapters and knowledge distillation into a compact BLIP-1 student. Achieved CIDEr +179% over baseline; student model runs at 4× lower GPU memory.
`Python` `PyTorch` `HuggingFace Transformers` `LoRA` `Knowledge Distillation` `CLIP`

**[AI Piano Music Analysis](https://github.com/TungAnhNguyen5/final_419_project)** — Research project analyzing AI-generated piano music (MuseNet, Bach Doodle, MusicLM) for harmonic structure, key consistency, and melodic coherence using the MAESTRO dataset and `music21`.
`Python` `music21` `mido` `pandas` `seaborn`
